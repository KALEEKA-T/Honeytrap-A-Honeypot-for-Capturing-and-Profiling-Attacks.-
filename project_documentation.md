# Project Documentation 

 

## Introduction 

Cyber attacks are increasing rapidly across internet-facing systems. Honeypots are security mechanisms designed to attract attackers and study their behavior. 

 

## Project Description 

This project deploys the Cowrie SSH honeypot to simulate a vulnerable system. The honeypot captures attacker interactions such as login attempts and executed commands. 

 

The collected logs are forwarded to Elasticsearch using Filebeat. Kibana dashboards are then used to visualize and analyze the attack data. 

 

## Key Visualizations 

- Attack timeline 

- Top attacker IP addresses 

- Geographic distribution of attacks 

- Log source distribution 

 

## Benefits 

- Real-time attack monitoring 

- Better understanding of attacker behavior 

- Centralized log analysis 

- Improved threat intelligence
