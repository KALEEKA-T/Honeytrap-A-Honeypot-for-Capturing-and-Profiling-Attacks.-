# Honeytrap-A-Honeypot-for-Capturing-and-Profiling-Attacks.-
A honeypot-based cyber attack monitoring system using the Cowrie SSH honeypot integrated with the ELK Stack (Elasticsearch, Filebeat, and Kibana). The project captures attacker interactions, analyzes honeypot log data, and visualizes attack patterns through real-time dashboards for cybersecurity monitoring and threat intelligence.
