# Honeypot-Based Cyber Attack Monitoring using ELK Stack 

 

## Project Overview 

This project implements a honeypot-based cyber attack monitoring system using the Cowrie honeypot integrated with the ELK Stack (Elasticsearch, Logstash/Filebeat, and Kibana). The system captures attacker interactions such as login attempts and executed commands and visualizes the data using Kibana dashboards. 

 

## Objectives 

- Capture attacker activities using a Cowrie SSH honeypot 

- Forward honeypot logs using Filebeat 

- Store and index logs using Elasticsearch 

- Visualize attack patterns using Kibana dashboards 

 

## Technologies Used 

- Cowrie Honeypot 

- Elasticsearch 

- Filebeat 

- Kibana 

- Ubuntu Linux 

 

## System Architecture 

The system architecture consists of four main components: 

 

1. Cowrie Honeypot – captures attacker interactions 

2. Filebeat – collects and forwards logs 

3. Elasticsearch – stores and indexes logs 

4. Kibana – visualizes attack patterns 

 

## Features 

- Real-time attack monitoring 

- Visualization of attack frequency 

- Identification of top attacker IP addresses 

- Geographic mapping of attack sources 

- Analysis of attacker commands
